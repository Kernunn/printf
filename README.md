# printf

Recode printf from libc

### Prototype

``int ft_printf(const char *, ...)``

### Usage

``make`` create ``libprintf.a`` and ``libft.a``

Supports the following:

* Type field: %, c, p, d, i, u, x, X, f, g, e, n
* Length field: l, ll, h, hh
* Precision field
* Width field
* Flags field: -, +, space, 0, #

For usage information: man 3 printf
